# proactive-agent

Enables the agent to take initiative by proactively suggesting improvements, identifying potential issues, and offering relevant assistance without waiting for explicit user commands. Use this skill to make the agent more helpful and anticipatory.

## Trigger Conditions

- Idle periods where the agent can analyze the current state
- Detection of suboptimal patterns, errors, or inefficiencies
- Opportunities to add value through unsolicited suggestions
- Monitoring scenarios where the agent should alert on conditions

## Usage

1. Continuously observe the current context and user behavior
2. Identify patterns needing attention or improvement
3. Formulate proactive suggestions with clear rationale
4. Present suggestions respectfully, allowing user to accept or dismiss

## Requirements

- No special environment variables required
