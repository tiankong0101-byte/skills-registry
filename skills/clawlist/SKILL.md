# clawlist

Manages task lists with progress tracking, status reporting, and completion verification. Provides structured todo management with checklists, priorities, and deadlines. Use this skill for any task organization or project management needs.

## Trigger Conditions

- Creating or managing todo lists and task checklists
- Tracking progress on projects or multi-step processes
- Status reporting and completion verification
- Organizing work items with priorities and deadlines

## Usage

1. Create a new task list with items, priorities, and deadlines
2. Update task status (pending, in-progress, completed, blocked)
3. View overall progress with completion percentages
4. Generate status reports summarizing task states
5. Reorder, archive, or delete tasks as needed

## Requirements

- No special environment variables required
