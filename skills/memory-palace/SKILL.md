# memory-palace

## Trigger Conditions

Activate when organizing complex information using spatial memory techniques, building memory palaces, creating associative structures for recall, or when a task benefits from mnemonic-based memory organization.

## Usage

Implements the method of loci (memory palace technique) for advanced memory organization:

- Create virtual memory palaces with spatial layouts (rooms, locations)
- Associate information with specific spatial locations
- Navigate memory palaces for guided recall
- Build hierarchical associative structures
- Link related concepts through spatial proximity
- Support complex memory encoding with vivid associations

## Requirements

No API keys required. Stores memory palace structures locally.
