# evo-benchmark

Benchmarking and capability measurement skill. Use when measuring system performance or comparing iterations.

## Trigger Conditions

- Performance testing before/after changes
- Capability assessment to identify strengths and weaknesses
- Tracking metrics over time for regression detection
- Comparing different approaches or configurations

## Usage

1. Define clear, measurable benchmarks (execution time, success rate, token usage, etc.)
2. Create reproducible test scenarios with controlled inputs
3. Run the baseline measurement, then the candidate measurement
4. Report results with statistical significance (mean, median, percentiles)
5. Highlight regressions or improvements with actionable recommendations

## Requirements

No special environment variables required.
