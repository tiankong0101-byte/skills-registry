# gbrain

## Trigger Conditions

Activate when the task involves building or querying a knowledge graph, mapping relationships between concepts, organizing information into graph structures, or performing graph-based reasoning and memory retrieval.

## Usage

Implements a graph-based memory system where knowledge is stored as nodes (concepts) and edges (relationships). Use this skill for:

- Building structured knowledge graphs from unstructured information
- Querying relationships between concepts
- Traversing graph paths for associative reasoning
- Storing and retrieving contextual memories with graph connections
- Visualizing knowledge structures

## Requirements

No additional API keys required. Uses local storage for graph data persistence.
