# summarize

Content summarization skill. Use when condensing long documents, logs, or conversations into concise summaries.

## Trigger Conditions

- Long documents or articles need a brief overview
- Chat logs or execution transcripts need digest summaries
- Research findings need key point extraction
- Meeting notes or lengthy discussions need action item distillation

## Usage

1. Read the full content to understand context and structure
2. Identify the most important information — key findings, decisions, action items, critical data
3. Produce a summary that preserves all essential details while removing redundancy
4. Adapt summary length to the use case (one-line tl;dr, bullet list, paragraph, hierarchical)
5. Verify the summary is factually accurate and does not introduce distortion

## Requirements

No special environment variables required.
