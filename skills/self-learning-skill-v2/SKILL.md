# self-learning-skill-v2

## Trigger Conditions

Activate when the system needs to adapt its behavior based on past interactions, when skills should improve through usage patterns, or when implementing learning loops that refine skill performance over time.

## Usage

Enables skills to learn and adapt through experience:

- Track usage patterns and outcomes for skill refinement
- Adjust skill behavior based on historical success/failure
- Implement feedback loops for continuous improvement
- Learn user preferences and adapt responses accordingly
- Optimize skill selection and parameter tuning
- Maintain learning state across sessions

## Requirements

No API keys required. Uses local storage for learning state persistence.
