## 核心框架一：超级转化率漏斗模型

### 普通人 vs 专业人理解的转化率

| 理解层次 | 公式 | 问题 |
|---------|------|------|
| 普通理解 | 转化率 = 订单量 / UV | 无法定位问题在哪一步 |
| 专业理解 | 最终转化率 = 转化率1 × 转化率2 × 转化率3 × ... | 追踪每一步，找到短板 |

**核心洞察**：你需要追踪从广告展现到最终成交的全流程每一步的转化率，才能找到问题所在。

### 漏斗模型核心原则

1. **每一步转化率都不可能100%，每一步都会有流失**
2. **从广告展现到成交的全流程步骤越多，整体流失越大**
3. **提高整体转化率的方法**：
   - 减少从展现到成交的步骤
   - 优化每一步的转化率

### 案例：少儿英语培训5步漏斗

从拿到有意向家长的电话号码到成交，一共五步：

```
号码到接通 → 接通到筛选有效号码 → 有效号码到预约试听 →
预约试听到试听完 → 试听完到缴费报名
```

每一步都有具体的转化率和问题分析。通过精细化追踪，发现某月营业额过亿元的教育公司，营销管理水平其实存在大量漏洞。

## 核心框架二：下单三步法

消费者从看到商品到最终成交，三步：

```
第一步：激发兴趣
    ↓
第二步：建立信任（初级信任 → 中级信任 → 高级信任）
    ↓
第三步：立刻下单
```

---

## 核心框架三：陈勇转化六要素

陈勇转化六要素分别对应下单三步法的三个阶段：

| 要素 | 对应步骤 | 作用阶段 |
|------|---------|---------|
| **互惠** | 第一步：激发兴趣 | 产生联系 |
| **承诺与兑现** | 第二步：建立信任 | 初级信任 |
| **信任状** | 第二步：建立信任 | 中级信任（信任转嫁）|
| **畅销好评** | 第二步：建立信任 | 高级信任（从众心理）|
| **痛点刺激** | 第三步：立刻下单 | 远离痛苦 |
| **稀缺** | 第三步：立刻下单 | 立刻成交 |

### 六要素详解

#### 1. 互惠（激发兴趣）

**定义**：相互给对方好处，你给客户好处（如优惠价、赠品等），客户直接给你带来好处（如使用优惠券购买）

**核心原理**：中国古话"来而不往非礼也"——互惠让双方关系更紧密

**主要作用**：让消费者与商家产生联系，在下单三步第一步"激发兴趣"中产生作用

**次要作用**：在第三步"立刻下单"中，客户使用优惠券购买 = 互惠

**使用技巧**：
- 互惠是双赢
- 案例：商场买衣服导购使用互惠技巧（先给优惠、再给赠品）
- 优惠券使用技巧：把"确认"按钮改为"我确认使用此券"可提高使用率

#### 2. 承诺与兑现（轻度信任）

**定义**：做出承诺 + 兑现承诺的保障措施 = 让人相信承诺可以实现

**核心原理**：当众做出承诺 + 有兑现保障 = 信用建立

**主要作用**：让消费者对商家产生轻度信任，在下单三步第二步"建立信任"的"初级信任"阶段发挥作用

**使用技巧**：
- 承诺要具体、可验证
- 需要配套的保障措施
- 案例：朋友圈"不减10斤不换头像"——260个赞的压力让人坚持减肥

**商业应用**：
- 承诺与兑现的方法：先做出承诺，然后在页面其他地方提供兑现的保障措施
- 案例：创维电视详情页——承诺"屏幕不漏光"，然后说明检测流程

#### 3. 信任状（信任转嫁）

**定义**：自带信任的事物，把对信任状的信任转嫁到商品或服务上

**核心原理**："爱屋及乌"——明星代言就是把粉丝对明星的信任转嫁到产品

**主要作用**：信任转嫁，在下单三步第二步"建立信任"的"中级信任"阶段发挥作用

**信任状类型**：

| 类型 | 说明 | 案例 |
|------|------|------|
| 机构大/历史久 | 从事一个行业时间越长越让人信任 | 创维"专注电视领域31年" |
| 证件/认证/检测报告 | 权威机构出具 | OPPO"充电5分钟通话2小时"需要检测报告 |
| 专家背书 | 专业权威 | 世界知名花艺师 |
| 名人/明星代言 | 公众人物信任转嫁 | 明星代言 |
| 名人亲自使用 | "亲测有效" | 梅婷、李念、孙红雷等明星在爱帝宫坐月子 |
| 知名公司客户 | 大公司信任背书 | 世界500强定制鲜花供应商 |

**使用原则**：**大家都懂的信任状强于业内的信任状**
- 错误示范："职业九段领衔的师资团队"（业内才懂）
- 正确示范："棋圣聂卫平领衔的师资团队"（大众都懂）

**案例：花点时间**
- 投资人是高圆圆（明星）
- 世界500强定制鲜花供应商
- 挖掘名师弟子上：首席花艺师是Dior、Hermès等一线品牌发布会御用花艺师的弟子

#### 4. 畅销好评（从众心理）

**定义**：利用人喜欢跟从大众选择的本能，通过展示销量和好评来建立信任

**核心原理**：生物学告诉我们，人的基因有群居特性，喜欢跟从大多数人的选择

**主要作用**：产生从众心理，在下单三步第二步"建立信任"的"高级信任"阶段发挥作用

**使用形式**：

| 形式 | 说明 | 案例 |
|------|------|------|
| 绕地球×圈 | 销量足够大的直观表达 | 香飘飘"杯子连起来可绕地球一圈" |
| 全国××家店 | 分支机构多 | 无忧保姆"116家门店"地图展示 |
| 用户覆盖全国/全球 | 地域广泛 | 学员覆盖全国34个省区市 |
| ××人或××客户的选择 | 数字大 | 10万+用户的选择 |
| 销售×××万 | 直接销量 | 按销量排序选择 |
| 客户评价 | 真实口碑 | 包含头像、姓名、城市、标签、具体内容 |
| ×个月××人口碑见证 | 新产品众筹后 | 四个月17.6万用户口碑见证 |

**案例：无忧保姆**
- 全国门店地图展示（小红旗标城市）
- 评价板块包含：客户姓氏+头像+锦旗+城市+服务类型+好评标签+具体评价

#### 5. 痛点刺激（远离痛苦）

**定义**：找出客户的核心痛点，在决策关键时刻刺激痛点

**核心原理**：人天生对痛苦比快乐更敏感（损失厌恶），刺激痛点能让客户感受到"不买会痛苦"

**主要作用**：让客户产生"远离痛苦"的动力，推动决策

**使用技巧**：
- 需要把所有可能的情况都模拟一遍，找到对应的痛点刺激
- 案例：儿童围棋培训话术优化，预约率从33%提高到74%
- 关键：讲"雪中送炭"的卖点，而非"锦上添花"的卖点

#### 6. 稀缺（立刻成交）

**定义**：制造稀缺感，让客户觉得"现在不买就没了"

**主要作用**：推动客户立刻行动，消除拖延

**使用形式**：
- 限时优惠
- 限量供应
- 断码/最后一件
- 专属优惠（仅限本人）
