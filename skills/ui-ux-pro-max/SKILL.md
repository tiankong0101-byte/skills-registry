# ui-ux-pro-max

Specialized skill for professional-grade design system creation, component library development, and high-fidelity interactive prototyping. Use when the user needs a comprehensive, scalable design solution for production applications.

## Trigger Conditions

- User requests a complete design system or UI component library
- User needs a production-grade prototype with micro-interactions and animations
- User asks for design tokens, style guides, or pattern libraries
- Tasks involving design-to-code handoff with developer-ready specs

## Usage

1. **Audit & Research** — Review existing product, brand guidelines, and target platform conventions (Material Design, HIG, etc.).
2. **Design System Architecture** — Define design tokens (colors, typography, spacing, shadows, breakpoints). Create a component hierarchy (atoms → molecules → organisms).
3. **Component Design** — Design each component with all states (default, hover, active, disabled, error, loading). Include responsive variants.
4. **Accessibility** — Ensure WCAG 2.1 AA compliance for color contrast, focus indicators, keyboard navigation, and screen reader support.
5. **Prototyping** — Build interactive prototypes with transitions, micro-interactions, and gesture support. Cover happy path, edge cases, and error states.
6. **Documentation** — Produce a design system documentation with usage guidelines, code snippets, and implementation notes.
7. **Handoff** — Export design specs (Figma/HTML/CSS), assets, and a component usage guide for developers.

## Requirements

- No external API keys required.
- Output as structured markdown with HTML/CSS code examples or design specifications.
