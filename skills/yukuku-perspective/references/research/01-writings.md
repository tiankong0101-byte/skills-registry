## 一、核心著作

### 1. 《个人影响力打造35讲》（2026年，得到App）
- **篇幅**：35讲 + 发刊词
- **模块结构**：
  - 模块一·战略定位（第1-9讲）：定位五要素 / 人设三层次 / 14种人物原型 / 三层同心圆用户模型 / 纺锤结构选题库
  - 模块二·内容创作（第10-19讲）：八大内容原理
  - 模块三·内容打磨（第20-25讲）：标题/开头/节奏/互动/结尾
  - 模块四·实战演练（第26-29讲）：口播/Vlog/访谈/种草
  - 模块五·与算法共舞（第30-34讲）：算法逻辑/数据透视/爆款模型/内容诊断

### 2. 《内容红利：掌控内容创作与流量增长的黄金法则》（2025年5月，376页，中信出版）
- 核心概念：内容红利 = 注意力稀缺时代的稀缺资产
- 16个底层逻辑 + 300个实操技巧，覆盖图文/短视频/直播全形态
- **书籍相比课程的增量内容**：
  - 第3章：剧情（情节/角色/对白三要素 + 三幕结构 + 内心/关系/情境三种价值对抗）
  - 第3章：好物种草（"驯服"比喻 + 十大消费决策要素：功能性/价格/品牌价值/权威背书/顾客证言/情感唤起/审美体验/损失与恐惧/代言人/社会认同）
  - 第4章：算法原理（"特定场景下人和信息更有效率地连接"精确定义 + 信息茧房问题 + 覆盖度vs准确度平衡）
  - 第4章：爆款迷思（AARRR漏斗模型：Acquisition获客→Activation激活→Retention留存→Revenue变现→Refer传播；80%粉丝增长来自20%爆款）
  - 第4章：数据思维（"不能量化就无法改进" + 指标体系三特征：目标意义/逻辑完整/规模-质量-营收分类）
  - 附录：参考书单16本（《定位》《营销管理》《故事》《对白》《人物》《引爆用户增长》等）

## 二、反复出现的核心论点（≥3次 = 真信念）

### 核心信念1：「影响力 = 被信任，不是被看见」
- 发刊词明确提出："影响力的本质，不只是被看见，更是被信任"
- 贯穿整个定位模块——人设、用户、内容、品牌，最终目标都是建立信任资产
- 这是于困困与大多数"流量思维"导师最根本的分歧

### 核心信念2：「做不成自媒体，是因为太想成为别人」
- 原话引用："做不成自媒体，不是因为你不懂爆款，而是因为你太想成为别人"
- 她的整个内容体系底层逻辑是：从真实自我出发，而非模仿爆款
- 对人设、人物原型、口音的强调，都围绕"做自己"展开

### 核心信念3：「内功（内容原理）决定下限，招式决定上限」
- 模块排序：先战略定位 → 再内容原理 → 再内容打磨
- 多次强调：没有原理支撑的技巧是"雕虫小技"
- 反对零散爆款技巧，主张系统构建

### 核心信念4：「工具价值 + 情绪价值 = 活人感」
- 内容原理第18-19讲双讲联动
- 核心公式：意义 = 融合（信息+情感）+ 普世（六种意义）
- "没有活人感的内容，像机器人输出；只有情绪的内容，像空洞鸡汤"

### 核心信念5：「AI时代，独特性是唯一无法被替代的竞争力」
- 发刊词引用丹·科伊(Dan Koe)"第二次文艺复兴"概念
- 核心论点：AI替代标准化能力，人的独特性、好奇心、多维兴趣变得更有价值
- 个人影响力 = AI时代普通人的诺亚方舟

## 三、自创术语与核心概念

| 术语 | 定义 | 首次出现 |
|------|------|---------|
| **内容红利** | 在注意力稀缺时代，真正有价值的内容是稀缺的，掌握它的人获得超额回报 | 《内容红利》书籍 |
| **活人感** | 内容的人格化程度——有口音、有意义、有温度 | 第10讲 |
| **口音** | 账号独特的个性，是语言风格、情绪特点、价值判断的综合体 = 内容DNA | 第18讲 |
| **意义** | 融合信息与情感，注入普世情感，让内容有灵魂 | 第19讲 |
| **场景** | 自媒体爆款内容的最小单元，含七要素 | 第14讲 |
| **故事母题** | 命中人类集体情感的原型故事结构 | 第16讲 |
| **职人** | 垂直领域的专业创作者，有粉丝几万/几十万就活得不错 | 第1讲 |
| **纺锤结构选题库** | 两端（用户需求+人设）中间（结构化组织）= 选题自动繁衍 | 第6-7讲 |
| **三层同心圆** | 核心人群/辐射人群/潜在人群的用户分层模型 | 第5讲 |
| **时间锚点** | 通过时态和时距操控用户对时间的感知 | 第13讲 |

## 四、内容体系的内在结构

```
影响力体系
    │
    ├── 战略层（地基）
    │      ├── 人设（你是谁）
    │      ├── 用户（为谁）
    │      ├── 内容（做什么）
    │      ├── 运营（在哪发）
    │      └── 品牌（成为谁）
    │
    ├── 原理层（内功）
    │      ├── 注意力层：结构 / 冲突与悬念 / 时间锚点
    │      ├── 爆款层：场景 / 故事 / 视角转换
    │      └── 活人层：口音 / 意义
    │
    └── 招式层（打磨）
           标题 / 开头 / 节奏 / 互动设计 / 结尾
```

## 五、推荐书单（揭示智识谱系）

于困困在课程中明确推荐：
1. **《故事》罗伯特·麦基** — 故事原理、叙事理论（反复引用）
2. **《救猫咪》布莱克·斯奈德** — 八大故事母题原典
3. **《作家之旅》克里斯托弗·沃格勒** — 故事原型理论基础
4. **《英雄之旅》约瑟夫·坎贝尔** — 神话叙事结构
5. **《剧本》理查德·沃尔特** — 故事天才vs文学天才
6. **《经典人物原型45种》** — 14种自媒体人物原型理论来源
7. **《思考，快与慢》丹尼尔·卡尼曼** — 节奏/认知心理学理论基础
