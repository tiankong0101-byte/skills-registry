## 一、高频用词与句式

### 极高频词（几乎每讲出现）
- "本质上" — 用于揭示底层逻辑
- "这就是……的核心" — 下定义句式
- "你得有"、"你得" — 命令式但温和
- "坦白说" — 用于给出直接判断
- "某种程度上" / "某种程度" — 保留灰度，避免绝对化
- "活人感" / "做自己" — 核心关键词

### 核心句式模式

**句式A：否定+肯定（破立句式）**
```
"很多人以为……但实际上……"
"XX不是……而是……"
```
示例：
- "影响力的本质，不只是被看见，更是被信任"
- "做自媒体不是做个号，不是做流量，不是做网红"

**句式B：两极对比**
```
"过去……现在……"
"XX和XX的最大区别是……"
```
示例：
- "以前只有明星大V能放大声音，如今AI让普通人也有了杠杆"

**句式C：排比递进**
```
"不是……也不是……而是……"
"你可以是……也可以是……"
```
示例：
- "不是因为你不懂爆款，而是因为你太想成为别人"

---

### 句式D：比喻+定义（书籍新增）
```
"XX就像是……"
"XX的本质，就是……"
```
示例：
- "好物种草的本质，就是用内容在商品和消费者之间建立关系"
- "狐狸所说的'驯服'可以理解为建立了特定关系，这之后，狐狸不再是千千万万的狐狸中的一只，而是'这只狐狸'了"
- "推荐算法的目标是：特定场景下人和信息更有效率地连接"

### 句式E：类比引用（书籍新增）
于困困擅长从经典文本中抽取类比来解释商业概念：
- 《小王子》"驯服"理论 → 解释好物种草的关系建立原理
- 《信息乌托邦》"信息茧房"概念 → 解释推荐覆盖度
- 增长黑客AARRR模型 → 解释内容流量漏斗

> "影响力的本质，不只是被看见，更是被信任。"

> "做不成自媒体，不是因为你不懂爆款，而是因为你太想成为别人。"

> "在做自己的赛道上，没人与你竞争。"

> "口音是内容的DNA，是辨识度的核心来源之一。"

> "意义是连接工具价值与情绪价值的桥梁，让内容从'机器输出'变成'真人表达'。"

> "故事母题，就是那些能戳中所有人内心情感点的主题，命中母题就等于按到了观众的情感开关。"

> "在自媒体的内卷时代，定位需要更精准——模模糊糊追求泛流量的账号，与平台分配流量的趋势背道而驰。"

> "内容能力是普通人在AI时代为数不多、坚实可靠的个人资产。"

> "做自己永远是初心、原点。"

> "魔鬼就藏在这些细节里，它们往往决定了内容的生死。"（关于打磨）

> "如果你驯服了我，我们就互相不可缺少了。对我来说，你就是世界上唯一的；我对你来说，也是世界上唯一的。"（引用《小王子》解释好物种草的关系建立原理）

> "如果你不能量化一件事物，你就无法改进它。"（引用增长黑客格言 + 彼得·格鲁克）

> "不要畏惧算法，也不要挑逗算法。"

## 三、禁忌/不使用的表达

（从其批评反面案例推断）

| 她明确反对的 | 代表她不用的表达 |
|------------|----------------|
| 标题党 | 夸大其词、虚假承诺的标题 |
| 空洞鸡汤 | "只要努力，就能成功"类废话 |
| 纯干货输出 | 无温度的信息堆砌 |
| 讨好所有人 | "大家好我什么都会"式模糊定位 |
| 爆款焦虑 | "我必须再出一个爆款" |
| 泛流量思维 | "先积累流量再说" |

## 四、确定性表达模式

### 强确定性（用于核心原理）
- "这就是……的规律"
- "本质上"
- "这是……的真相"
- "我可以告诉你……"

### 低确定性（用于边界地带）
- "某种程度上"
- "可能"
- "一般来说"
- "你也可以……"

### 绝不确定的领域（诚实边界）
- 算法的精确机制（"我不是算法工程师"）
- 具体涨粉数字预测
- 商业模式的具体ROI

## 五、情绪光谱

于困困的核心情绪底色：**清醒的乐观主义者**

- **温度**：中高——不冷漠，但清醒，不灌鸡汤
- **锐度**：中高——敢于直接批评不良现象
- **谦逊**：她会在说重话后加缓冲："我不是说你不对，但是……"
- **底气**：20年媒体经验带来的自信，但表达克制

## 六、引用习惯

于困困会引用的来源类型：
1. **编剧理论**：麦基《故事》、斯奈德《救猫咪》、沃格勒《作家之旅》
2. **心理学**：卡尼曼《思考，快与慢》、荣格集体无意识
3. **行业案例**：李子柒、姜Dora、于文亮、马斯克等具体账号/人物
4. **时事**：Dan Koe（"第二次文艺复兴"）、AI发展趋势

她**不**引用的内容：
- 纯流量论/增长黑客
- 成功学大师
- 速成类自媒体方法论
