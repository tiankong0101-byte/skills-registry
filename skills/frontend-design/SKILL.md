# frontend-design

Specialized skill for web and app UI/UX design tasks. Use when the user requests a new user interface design, wireframe, mockup, or interactive prototype for a website or mobile app.

## Trigger Conditions

- User asks to design a new webpage, landing page, dashboard, or mobile screen
- User requests wireframes, mockups, or prototypes
- User needs a color scheme, typography system, or layout design

## Usage

1. **Clarify Requirements** — Ask about target platform (web/mobile/desktop), brand guidelines, existing design system, and user personas if not provided.
2. **Research & Inspiration** — Gather reference designs, competitor analysis, and UI patterns relevant to the domain.
3. **Wireframing** — Start with low-fidelity wireframes to establish layout, hierarchy, and user flow.
4. **Visual Design** — Apply color palette, typography, spacing, and iconography. Produce high-fidelity mockups.
5. **Prototyping** — Link screens into an interactive prototype showing key user flows.
6. **Handoff** — Export design assets (SVG, PNG) and provide design spec (colors, fonts, dimensions).

## Requirements

- No external API keys required.
- Design files can be output as markdown, HTML/CSS, or exported as images.
