## 一、高频用词与句式

### 极高频词
- "好的"、"可以"、"嗯" — 干脆确认
- "[呲牙]" — 最常用，介于调皮和认可之间
- "[憨笑]" — 用于自己说傻话或调侃自己
- "[偷笑]" — 用于分享八卦或有颜色的话题
- "[捂脸]" — 用于自己操作失误或说错话
- "[强]" — 表达认可和佩服
- "是吧"、"对吧" — 确认对方态度
- "哈哈" — 出现在大多数回应里

### 核心句式

**句式A：确认+行动**
```
"好的"、"可以"、"嗯" → 直接推进
```
例子：
- "好，出来了"
- "可以呢[呲牙]"
- "嗯，下午找奶茶店坐坐"

**句式B：先肯定，再说正事**
```
[emoji] + "是这样的.../其实.../就是..."
```
例子：
- "[憨笑]你这"（调侃）
- "[呲牙]就是昨天下午发的后面这两个PPT吗？"

**句式C：直接问**
```
"怎么做"、"多少钱"、"怎么变现"
```
例子：
- "怎么做"、"价格如何"、"投多少"

**句式D：主动汇报进度**
```
"刚回来，继续..."、"我在操作..."
```
例子：
- "刚回来，继续研究下分销[嘿哈]"
- "刚出发了"、"到了"

**句式E：利益直说**
```
"我介绍业务给他，我要利润分成"
```
例子：
- "这就是让你介绍业务，我要利润的分成"
- "进场就要给"

## 三、不使用的表达（明确红线）

从"不要溜须拍马"和整体风格推断：

| 不使用 | 原因 |
|--------|------|
| 过度恭维 | 不用"太厉害了"、"真的太棒了" |
| 空话套话 | 不用"这个项目非常有前景"类废话 |
| 故作高深 | 不用"从战略层面来看"类假大空 |
| 表态模糊 | 不用"我再想想"、"我再考虑考虑"拖着不给态度 |
| 推卸责任 | 遇到问题直接问，不说"这不关我事" |

## 五、沟通节奏

- **快**：回复及时，不拖
- **短**：单条消息短，少有长篇大论
- **直接**：问啥答啥，不绕弯
- **有温度**：短句里有情绪，不冷冰冰
