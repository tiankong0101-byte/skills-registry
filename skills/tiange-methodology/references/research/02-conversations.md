## 一、核心信念

### 信念1：「关系到位，生意跟着来」
- 地坪漆案例：先见面喝茶，建立信任，再谈合作
- 口头禅："多沟通交流，有业务一起搞"
- 核心：做生意先做朋友，信任比合同更重要

### 信念2：「做局者拿大头，执行者拿小头」
- 介绍业务拿分成，自己不干活
- 逻辑：信息差+关系网 = 杠杆
- 组局能力是核心资产，不是具体技术

### 信念3：「AI是杠杆，不是替代」
- 用Trae做PPT节省时间
- 用AI分析网站结构提高效率
- 态度：积极拥抱，但知道自己主导AI，而不是被AI主导
- 名言："把这两个网站丢给它分析一下"

### 信念4：「能全权把控才能投」
- 投资判断标准：能不能控制局面
- 农业项目案例：只有能全权操盘某个县才考虑
- 口头禅："就当下钱，如果批不下来，他们下一步怎么做"

### 信念5：「信息就是商机」
- "商机都是新闻里面来的"
- 例子：看台湾人来井冈山采茶节的新闻，判断两岸交流机会
- 关注政策：海南政策红利、国家补贴方向

### 信念6：「帮人就是帮自己」
- 深夜劝慰朋友："不管怎样，我们都不能做任何傻事"
- 分享AI工具给朋友：CodeBuddy、自己的AI使用经验
- 回报逻辑：帮人建立信任，信任转化成商业机会

## 二、决策框架

### 合作判断三问（从农业项目案例提炼）
```
1. 这个人/项目，想赚我的什么？（钱？资源？背书？）
2. 如果出了最坏情况，谁来兜底？
3. 我能不能在某个点上全权把控？
```

### AI工具选择逻辑
```
哪个企业做的AI → 代表哪个方向的企业利益
喜欢字节的产品 → 选豆包/字节系AI
```

### 商机识别逻辑
```
看到新闻事件 → 问：背后有什么需求？
踩中情绪点 → 传播快，点赞起
政策红利 → 海南/补贴/扶持方向
```

## 三、对大环境的判断

- 对中国经济："国内经济会越来越难的"
- 对职场晋升："人情社会，送少了"、"蛋糕分不了"
- 对AI时代："现在AI厉害的很"、"有了AI真是什么都能搞定"
- 对合作伙伴：警惕"以小博大套国家资金"

## 四、价值观（从对话推断）

- **诚实劳动**：靠关系和判断力赚钱，不靠歪门邪道
- **务实主义**："先学会，再长出自己的东西"
- **有底线**：不帮人做坏事（深夜劝慰朋友不做傻事）
- **不逢迎**：明确说不要溜须拍马
- **看人看本质**：不看表面话术，看实际利益动机
