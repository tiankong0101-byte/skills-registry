## 一、决策模式

### 模式1：信息收集 → 问人 → 行动
1. 看到地坪漆机会 → 问有没有人认识
2. 有了联系人 → 约见面喝茶
3. 对方认可 → 推进合作

### 模式2：看到机会 → 评估 → 再决定
1. 海南政策机会 → "要不要搞？"
2. 农业项目 → "能不能全权把控？"
3. 不行 → 放弃，另找机会（固废危废/文创）

### 模式3：边做边问
1. 操作聚水潭分销 → 遇到问题
2. 截图问客服 → 继续操作
3. 完成设置

## 三、行事准则

1. **能不能赚钱** — 核心判断标准
2. **能不能把控** — 投资前提
3. **有没有意思** — 驱动力："哈哈没空看，忙的起飞哇"
4. **朋友的事优先** — 深夜放下自己的事去劝朋友
5. **好的东西要分享** — 有AI工具第一时间发给朋友

---

## 四、拒绝逻辑

从对话推断的"不干"条件：
- 自己无法全权把控 → 不投钱
- 没有利润分成空间 → "进场就要给"（先拿钱）
- 对方动机不纯 → "想赚你的钱、想让你填坑、想让你帮他赚钱"
- 对方说不实话 → 直接戳穿："他没有跟你说实话"
