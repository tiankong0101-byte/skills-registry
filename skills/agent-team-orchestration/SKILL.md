# agent-team-orchestration

Orchestrates teams of specialized agents to complete complex workflows. Assigns roles, schedules tasks, manages dependencies, and consolidates outputs from multiple agents. Use this skill when a single agent is insufficient and a team-based approach is needed.

## Trigger Conditions

- Multi-disciplinary tasks requiring different expertise areas
- Workflows with parallel and sequential dependencies
- Large-scale projects that benefit from分工 collaboration
- Any scenario where agent team formation improves efficiency

## Usage

1. Analyze the overall goal and decompose into agent-specific roles
2. Assign tasks to appropriate agent types
3. Manage inter-agent dependencies and handoffs
4. Monitor progress across the agent team
5. Synthesize results from all agents into a unified output

## Requirements

- No special environment variables required
