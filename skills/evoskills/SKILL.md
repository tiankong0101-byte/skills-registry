# evoskills

## Trigger Conditions

Activate when managing the skill ecosystem — analyzing skill quality, identifying improvement opportunities, evolving existing skills, or performing automated skill enhancement across the registry.

## Usage

This skill handles the continuous improvement lifecycle of the skill ecosystem. Use it to:

- Analyze skill usage patterns and quality metrics
- Identify outdated or underperforming skills
- Propose and apply automated improvements to skill definitions
- Track skill evolution history and versioning
- Manage skill dependencies and compatibility

## Requirements

No additional API keys required. Works with the local skill registry structure.
